# 凑十爆破 · Make-10 Blast

> 「以为很简单 结果尴尬了」—— 抖音同款数字消除小游戏

扫雷盘面 + 连连看玩法的合体。单文件、零依赖，手机/电脑浏览器打开 `index.html` 即玩。

## 玩法
1. 盘面像扫雷一样盖满方块，每格底下藏着一个数字 (1–9)，开局翻开中心 3×3。
2. 选两个**相加等于 10** 的数字（5+5、9+1、3+7…）把它们连起来。
3. **连接路径要畅通**：中间被别的数字挡住就连不上（连连看规则，最多拐两个弯；盖住的格子也算墙）。例如 9 和 1 之间隔着 6 和 5，就连不成。
4. 连接成功 → 两个数字**爆破消失**，并炸开它们**周围 8 格**（含四个斜角），露出新数字。
5. 把**所有盖住的格子全部翻开** = 挑战成功！

## 操作
- 点一个数字选中，再点另一个数字尝试连接；也可以直接从一个数字**拖到**另一个。
- 💡 提示：高亮一组当前可消的数字。
- 🔀 打乱：把场上的数字重排（死局救场）。
- 🔁 重开：换一局新盘面。

## 设计 / 验证
- 连连看寻路：BFS（带方向 + 拐弯计数，≤2 弯），盖住格与其他数字均视为墙。已通过 12 项路径规则单测。
- 盘面生成：全盘填入平衡的互补数字对，并保证开局至少有一对相邻可消，避免开局即死局。
- 可解性：2000 局**纯随机走法**模拟，胜率 92.5%；允许「打乱」救场后 99.4%。难度刚好——粗心会卡住，动脑能通关。
