# 跃动小子 · Jumping Kid

> 「你厉害你来把第二关过了！！！」

一个抖音同款的高难度精准跳跃小游戏（rage platformer），单文件、零依赖、手机/电脑都能玩。直接用浏览器打开 `index.html` 即可。

## 玩法
- **手机**：左下方向键移动，右下「跳」键跳跃（支持二段跳）。
- **电脑**：`←` `→` 移动，`空格` / `↑` 跳跃，`R` 重来本关。

## 关卡
1. **热身** —— 教你基本操作。
2. **魔鬼第二关** —— 移动平台、旋转锯齿、尖刺深坑、落石。每一跳都要刚刚好。
3. **凯旋** —— 收尾庆祝。

通关后会显示总用时与死亡次数，发到评论区比比谁是真大神。

## 特性
- HTML5 Canvas + 原生 JS，无外部依赖。
- 自适应缩放、刘海屏安全区适配、触控/键盘双输入。
- 手感细节：土狼时间（coyote time）、跳跃缓冲、二段跳、可变跳跃高度、挤压拉伸动画、粒子与镜头震动。
